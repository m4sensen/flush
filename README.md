# Flush

Flush is a modular, Tailwind-inspired Bash toolkit designed to make building simple Command-Line Interfaces (CLIs).

To get started, navigate to the `scripts/` directory and begin customizing `main.sh`.

# ✨ Features

- Tailwind Colors: Use 24-bit TrueColor names like emerald, amber, and sky.
- Smart Logging: Clean, consistent functions for Success, Error, Warning, and more.
- Auto-Centering: A center_text tool that handles multi-line ASCII art and ANSI colors.
- Feature Toggles: Easily turn off colors or emojis in config/lib.cfg for plain-text environments and more.

# 🚀 Getting Started

1. Installation

Clone the repository and source the toolkit in your script:
Bash

```bash
git clone https://github.com/m4sensen/flush.git
source flush/toggle.sh
```

2. Usage

Go to scripts/main.sh and start editing.

# 🏗 Requirements

- Bash 4.0+
- Terminal that supports TrueColor (24-bit) for the full color experience.
